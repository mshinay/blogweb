# 前后端对齐接口文档

> 统一响应字段：本文档统一使用 `code`、`message`、`data`；列表接口 `data` 统一为 `{ total, records }`。

# 用户模块

## 1. 接口名称
用户登录

## 2. 请求方式
POST

## 3. 请求路径
`/api/users/login`

## 4. 请求参数

### Body 参数（用于 POST/PUT）
```json
{
  "username": "zhangsan",
  "password": "123456"
}
```

## 5. 响应结构
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1,
    "username": "zhangsan",
    "nickname": "张三",
    "avatarUrl": "https://cdn.example.com/avatar/1.png",
    "role": 1,
    "status": 1,
    "jwtToken": "eyJhbGciOiJIUzI1NiJ9..."
  }
}
```

## 6. 字段说明
| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| id | number | 用户 ID |
| username | string | 用户名 |
| nickname | string | 昵称 |
| avatarUrl | string | 头像地址 |
| role | number | 角色，0=管理员，1=普通用户 |
| status | number | 状态，0=禁用，1=正常 |
| jwtToken | string | 登录态 token |

# 文章模块

## 1. 接口名称
获取文章列表

## 2. 请求方式
GET

## 3. 请求路径
`/api/articles`

## 4. 请求参数

### Query 参数（用于 GET）
| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| categoryId | number | 否 | 分类 ID |
| tagId | number | 否 | 标签 ID |
| authorId | number | 否 | 作者 ID |
| slug | string | 否 | 文章 slug |
| keyword | string | 否 | 标题/摘要/正文关键字 |
| page | number | 是 | 页码，从 1 开始 |
| pageSize | number | 是 | 每页条数 |

## 5. 响应结构
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "total": 2,
    "records": [
      {
        "id": 101,
        "title": "Spring Boot 实战",
        "slug": "spring-boot-guide",
        "summary": "一篇入门到实战的总结",
        "coverUrl": "https://cdn.example.com/article/101.png",
        "authorId": 1,
        "authorName": "zhangsan",
        "categoryId": 10,
        "categoryName": "Java",
        "categorySlug": "java",
        "tagList": [
          {
            "id": 1001,
            "name": "Spring Boot",
            "slug": "spring-boot"
          }
        ],
        "publishTime": "2026-03-20T10:00:00",
        "isTop": 1,
        "viewCount": 120,
        "commentCount": 8
      }
    ]
  }
}
```

## 6. 字段说明
| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| total | number | 总记录数 |
| records | array | 当前页文章列表 |
| records[].id | number | 文章 ID |
| records[].title | string | 标题 |
| records[].slug | string | 文章 slug |
| records[].summary | string | 摘要 |
| records[].coverUrl | string | 封面图地址 |
| records[].authorId | number | 作者 ID |
| records[].authorName | string | 作者名 |
| records[].categoryId | number | 分类 ID |
| records[].categoryName | string | 分类名称 |
| records[].categorySlug | string | 分类 slug |
| records[].tagList | array | 标签列表 |
| records[].tagList[].id | number | 标签 ID |
| records[].tagList[].name | string | 标签名称 |
| records[].tagList[].slug | string | 标签 slug |
